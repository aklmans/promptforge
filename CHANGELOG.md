# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project follows Semantic Versioning.

## [0.1.0] - 2026-04-12

### Added

- Initial PromptForge CLI and Ink TUI experience
- Prompt enhancement flow with provider/model configuration
- History, feedback iteration, diff view, and command palette
- Responsive TUI layout for narrow terminal sizes
- Unit tests for config parsing, LLM response normalization, and text utilities
- Release scripts for lint, typecheck, test, build, and dry-run packaging

### Changed

- Improved CLI argument validation and runtime error messages
- Hardened config parsing and provider/model resolution errors
- Tightened npm publish preparation and local config ignore rules
